# Kaiju Quality Consulting
## Kaiju Quality Examples Repo

The purpose of this repo is to provide basic templates for architecting, developing, and maintaining Web based test 
solutions. Within the provided project folders, you will see some basic examples on how to start a new web based
testing frameworks. The concepts applied in these are a sure way to kick off your own automated testing solution.

This
in no way a fully developed and scalable test solution. Rather, it is a backbone for you to grow and build.
